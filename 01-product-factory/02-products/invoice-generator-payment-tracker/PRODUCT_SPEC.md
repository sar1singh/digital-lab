# Product Spec — Nivora Invoice Generator & Payment Tracker

## Overview

A Google Sheet / Excel spreadsheet that helps freelancers, consultants, and independent professionals create invoices, track client information, monitor payments, and view outstanding balances.

## Format

- Google Sheet (copy-ready template)
- XLSX download
- PDF quick-start guide (1 page)

## Target Buyer

Freelancers, consultants, developers, designers, agencies, and independent professionals who need a simple system to create invoices and track payments without accounting software.

## Suggested Price

$7 launch price.

## Tabs

| # | Tab | Purpose |
|---|-----|---------|
| 1 | Start Here | Setup instructions, warnings, quick start |
| 2 | Dashboard | Key metrics: revenue, outstanding, invoice/payment status breakdowns |
| 3 | Clients | Client directory with contact info and status |
| 4 | Invoice Register | All invoices — sent, paid, overdue, draft |
| 5 | Payment Tracker | All payments received linked to invoices |
| 6 | Invoice Generator | Printable invoice layout with auto-calculations |
| 7 | Settings | Currency, default tax rate, status dropdown values |

## Features

- Dashboard with revenue summary, invoice status breakdown, top clients
- Client directory with status tracking
- Invoice register with auto-status (Paid, Overdue, Partial, Sent, Draft)
- Payment tracking linked to invoices
- Printable invoice generator with auto-calculated totals
- Settings tab for currency, tax rate, and dropdown values
- Automatic calculations — no manual formulas needed
- Formula-driven only — no macros, VBA, Apps Script, or external integrations

## Scope

- No SaaS
- No app
- No backend
- No macros
- No VBA
- No APIs
- No external integrations
- Single file product
