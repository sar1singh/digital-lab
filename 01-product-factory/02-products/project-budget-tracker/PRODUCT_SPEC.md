# Product Spec — Nivora Project Budget Tracker

## Overview

A Google Sheet / Excel spreadsheet that helps freelancers, agencies, and project managers track budgets across multiple projects — monitor spending vs budget, log expenses, track income, and view project profitability at a glance.

## Format

- Google Sheet (copy-ready template)
- XLSX download
- PDF quick-start guide (1-2 pages)

## Target Buyer

Freelancers, small agencies, consultants, project managers, and independent professionals who manage multiple projects and need to track budgets, expenses, and profitability without accounting software.

## Suggested Price

$7 launch price.

## Tabs

| # | Tab | Purpose |
|---|-----|---------|
| 1 | Start Here | Setup instructions, warnings, quick start |
| 2 | Dashboard | Key metrics: total budget, total spent, remaining, % used, profit, project status breakdown |
| 3 | Projects | Project list with budget, actual spend, remaining, % used, status |
| 4 | Expenses | Individual expense entries linked to projects |
| 5 | Income | Revenue/income entries linked to projects |
| 6 | Categories | Expense categories with descriptions |
| 7 | Settings | Currency, project status values, payment status values |

## Features

- Dashboard with budget KPIs, spending by category, project budget status, monthly spending trend
- Project list with auto-calculated budget vs actual tracking
- Expense logging linked to projects with category selection
- Income/revenue tracking per project
- Auto-calculated remaining budget and % used per project
- Status formulas: On Budget, Over Budget, Completed, On Hold
- Net profit calculation (Income - Expenses)
- Settings tab for currency and dropdown values
- Automatic calculations — no manual formulas needed
- Formula-driven only — no macros, VBA, Apps Script, or external integrations

## Scope

- No SaaS
- No app
- No backend
- No macros
- No VBA
- No APIs
- No external integrations
- Single file product
