# Nivora Project Budget Tracker — Quick-Start Guide

## Getting Started

1. **Open in Google Sheets**: Click the copy link to create your own editable copy in Google Drive. OR open the XLSX file in Microsoft Excel.

2. **Start Here tab**: Read the setup instructions on the first tab.

3. **Settings tab first**: Set your currency symbol and preferred dropdown values.

4. **Add projects**: Go to the Projects tab and enter your project details and budgets.

5. **Log expenses**: Use the Expenses tab to record spending against each project.

6. **Track income**: Use the Income tab to record revenue per project.

7. **View dashboard**: The Dashboard tab shows your budget KPIs, spending breakdown, and project profitability.

## Important Notes

- **Do not edit gray cells** — these contain formulas that calculate automatically.
- **Edit white/green cells** — these are input cells where you enter your data.
- **Sample data is included** — replace it with your own data. Delete or overwrite sample rows.
- **Backup your data** — keep a copy of your spreadsheet as you add data.
