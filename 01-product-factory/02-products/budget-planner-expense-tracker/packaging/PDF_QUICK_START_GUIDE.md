# Budget Planner & Expense Tracker — Quick Start Guide

---

## What You Receive

- **Google Sheet** — copy to your Google Drive using the /copy link
- **XLSX file** — open in Microsoft Excel or any spreadsheet app
- **PDF guide** — this document

---

## How to Use the Google Sheet Copy Link

1. Click the /copy link you received after purchase
2. Sign in to your Google account
3. Click **Make a copy**
4. The spreadsheet is saved to your Google Drive
5. Open it from Google Drive anytime

---

## How to Use the XLSX Version

1. Download the XLSX file from your purchase receipt
2. Open with Microsoft Excel, LibreOffice Calc, or Apple Numbers
3. Enable editing if prompted by Excel

---

## Which Cells to Edit (Green Background)

Edit these cells only:
- **Monthly Budget** → Planned Amount column (column C)
- **Expense Tracker** → all white cells (Date, Category, Description, Payment Method, Planned/Unplanned, Amount, Notes)
- **Bills Tracker** → all white cells
- **Debt Tracker** → Current Balance, Extra Payment (white cells only)
- **Savings Tracker** → Current Amount, Monthly Contribution (white cells only)

---

## Which Cells NOT to Edit (Gray Background)

Do not edit cells with gray background. These contain formulas that calculate automatically:
- Actual Amount and Difference columns in Monthly Budget
- Month column in Expense Tracker
- Payoff Progress % in Debt Tracker
- Progress % in Savings Tracker
- All Dashboard summary cards

Editing these will break the spreadsheet.

---

## How to Reset Sample Data

1. Go to **Expense Tracker** tab
2. Select all rows with sample data (rows 2-19)
3. Right-click → **Delete selected rows**
4. Repeat for Bills Tracker, Debt Tracker, and Savings Tracker

Or make a fresh copy of the original file to start clean.

## Basic Troubleshooting

| Problem | Solution |
|---------|----------|
| Dashboard shows #DIV/0! | Enter at least one income amount in Monthly Budget |
| Expense Tracker Month column is blank | Enter a date in column A (Date) |
| Formula shows #REF! | You may have deleted a referenced sheet. Re-download the original file |
| Dropdown not appearing | The cell may have been overwritten. Delete the cell content and try again |
| Numbers show as $0.00 | Enter actual amounts in the Planned column of Monthly Budget |
