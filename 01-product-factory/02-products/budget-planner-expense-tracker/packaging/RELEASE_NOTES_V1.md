# Release Notes — v1.0

## Product

Budget Planner & Expense Tracker

## Version

v1.0 — Initial release

## Included Tabs

| # | Tab | Description |
|---|-----|-------------|
| 1 | Start Here | Setup instructions, currency note, warnings, checklist |
| 2 | Monthly Budget | Planned vs actual income, expenses, and savings by category |
| 3 | Expense Tracker | Log every expense with date, category, payment method, and planned/unplanned flag |
| 4 | Bills Tracker | Track recurring bills with due dates, frequency, and paid/unpaid status |
| 5 | Debt Tracker | Monitor debt payoff across multiple debts with auto-calculated progress % |
| 6 | Savings Tracker | Set savings goals and track progress with auto-calculated % |
| 7 | Dashboard | Summary cards, category spending breakdown, expense pie chart, income vs expenses bar chart |

## Formats

- Google Sheet (copy link)
- Microsoft Excel (XLSX)
- PDF quick-start guide

## Known Limitations

- No compound interest modeling in Debt Tracker
- No transaction import or bank sync
- Charts are Excel-native; may need recreation in Google Sheets after upload
- Full-column SUMIF references may cause slower performance in Excel with very large (10k+ row) datasets
- No multi-currency support (single currency per sheet)
- No VBA macros or automation
- Static template — no auto-save or cloud sync beyond Google Drive

## Recommended Buyer Use Case

Individuals, freelancers, and solopreneurs who want a simple all-in-one budgeting spreadsheet without paying for a monthly subscription. Best for monthly budget planning, expense tracking, bill management, debt payoff monitoring, and savings goal tracking.
