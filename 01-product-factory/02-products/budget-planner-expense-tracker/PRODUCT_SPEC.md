# Product Spec — Budget Planner & Expense Tracker

## Overview

A Google Sheet / Excel spreadsheet that helps individuals, freelancers, and solopreneurs plan monthly budgets, track expenses, manage bills, pay down debt, and save toward goals.

## Format

- Google Sheet (copy-ready template)
- XLSX download
- PDF quick-start guide (1 page)

## Target Buyer

Individuals, freelancers, and solopreneurs who want a simple all-in-one budgeting system without learning complex finance software.

## Suggested Price

$7 launch price.

## Tabs

| # | Tab | Purpose |
|---|-----|---------|
| 1 | Start Here | Setup instructions, warnings |
| 2 | Monthly Budget | Income vs planned vs actual spending |
| 3 | Expense Tracker | Log every expense with category and date |
| 4 | Bills Tracker | Track recurring bills and payment status |
| 5 | Debt Tracker | Monitor debt payoff progress |
| 6 | Savings Tracker | Track savings goals |
| 7 | Dashboard | Summary view with key metrics |

## Features

- Monthly budget with planned vs actual comparison
- Expense logging with categories
- Bills due tracking with paid/unpaid status
- Debt payoff tracker with progress %
- Savings goal tracker with target dates
- Dashboard with total income, expenses, savings rate, top categories
- Automatic calculations — no manual formulas needed

## Scope

- No SaaS
- No app
- No backend
- No code
- Single file product
