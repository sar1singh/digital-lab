Quick Start Guide

What's Included

  • Excel workbook (.xlsx) — works with Microsoft Excel and Google Sheets
  • Google Sheets copy link — make a copy and use online
  • Release notes — summary of all features

Google Sheets Version

  1. Open the copy link provided in google-sheets-copy-link.txt
  2. Click File > Make a copy to save your own version
  3. Start from the Start Here tab
  4. Use the sheet tabs at the bottom to navigate between sections

Excel Version

  1. Open budget-planner-expense-tracker.xlsx in Microsoft Excel
  2. If you see a yellow security bar, click Enable Editing
  3. Start from the Start Here tab
  4. Use the sheet tabs at the bottom to navigate between sections

Start Here

Your home screen with setup instructions, navigation guide, checklist, currency note, and important warnings about formula cells.

Monthly Budget

Plan your monthly income and expenses:

  • Enter Planned Amount for each income and expense category
  • Actual Amount pulls automatically from Expense Tracker entries
  • Difference column shows over/under budget (green = under, red = over)
  • Totals update automatically

Expense Tracker

  • Enter the Date — Month fills automatically
  • Select a Category from the dropdown
  • Select Payment Method from the dropdown
  • Mark as Planned or Unplanned
  • Category totals update in Monthly Budget and Dashboard

Bills Tracker

  • Track bills with due dates, amounts, and frequency
  • Update Paid Status as you pay
  • Unpaid bills are highlighted in red
  • Dashboard shows total bills due

Debt Tracker

  • Enter starting balance, current balance, interest rate
  • Payoff Progress percentage updates automatically
  • Total debt remaining shows on Dashboard

Savings Tracker

  • Set goals with target amount and target date
  • Progress percentage updates automatically
  • Average savings progress shows on Dashboard

Dashboard

Your financial overview at a glance:

  • Total Income — planned income from all sources
  • Total Expenses — actual spending from Expense Tracker
  • Net Savings — income minus expenses
  • Savings Rate — percentage of income saved
  • Bills Due — unpaid bills requiring attention
  • Debt Remaining — total outstanding debt
  • Savings Progress — average progress toward goals
  • Category Breakdown — spending by category with charts

Common Notes

  • Cells with gray background contain formulas — do not edit them
  • Only edit white or green-background input cells
  • Keep category names consistent across tabs for formulas to work
  • Clear sample data before entering your own information
