# Screenshots — Manual Capture Guide

These screenshots should be captured manually from the Google Sheets version of the
Budget Planner & Expense Tracker.

## Required Screenshots

| File | What to capture |
|------|----------------|
| `01-dashboard-overview.png` | Full Dashboard tab showing summary cards, category breakdown, and charts |
| `02-monthly-budget.png` | Monthly Budget tab showing income and expense sections with planned vs actual |
| `03-expense-tracker.png` | Expense Tracker tab showing sample data, dropdown menus, and categories |
| `04-bills-debt-savings.png` | Combined view or three separate shots of Bills Tracker, Debt Tracker, and Savings Tracker |
| `05-product-bundle.png` | Collage or single image showing the workbook (Start Here + all tabs) |

## Capture Guidelines

- Use **1920 × 1080** or similar widescreen resolution
- Capture the **full window** showing the Google Sheets UI
- Ensure **sample data** is visible (demonstrates the product works)
- Avoid capturing personal or sensitive information
- Use **light mode** for clean, professional screenshots

## Process

1. Open the workbook in Google Sheets (or Excel with sample data loaded)
2. Navigate to each tab and arrange the view
3. Use Windows Snipping Tool, Greenshot, or similar
4. Save each file with the exact filenames listed above
5. Place the files in `delivery/screenshots/`

## Placement

Save all captured screenshots directly in:
`delivery/screenshots/`
