# Budget Planner & Expense Tracker — Quick Start Guide

## What's Included

- **Excel workbook** (.xlsx) — works with Microsoft Excel and Google Sheets
- **Google Sheets copy link** — make a copy and use online
- **Release notes** — summary of all features

## How to Use the Google Sheets Version

1. Open the copy link provided in `google-sheets-copy-link.txt`
2. Click **File > Make a copy** to save your own version
3. Start from the **Start Here** tab
4. Use the sheet tabs at the bottom to navigate between sections

## How to Use the Excel Version

1. Open `budget-planner-expense-tracker.xlsx` in Microsoft Excel
2. If you see a yellow security bar, click **Enable Editing**
3. Start from the **Start Here** tab
4. Use the sheet tabs at the bottom to navigate between sections

## Start Here

The Start Here tab is your home screen. It includes:
- Setup instructions
- Navigation guide listing all tabs
- A setup checklist with steps to follow
- Important notes about formula cells (gray background = do not edit)
- Currency format instructions

## Monthly Budget

Plan your monthly income and expenses:

1. Enter your **Planned Amount** for each income category (Salary, Freelance, Side Hustle)
2. Enter your **Planned Amount** for each expense category (Housing, Utilities, Groceries, etc.)
3. The **Actual Amount** column pulls automatically from your Expense Tracker entries
4. The **Difference** column shows if you are over or under budget (green = under, red = over)
5. Totals update automatically at the bottom of each section

## Expense Tracker

Log every expense:

1. Enter the **Date** — the Month column fills automatically
2. Select a **Category** from the dropdown
3. Add a **Description**
4. Select **Payment Method** from the dropdown
5. Mark as **Planned** or **Unplanned**
6. Enter the **Amount**
7. The category total updates automatically in your Monthly Budget and Dashboard

## Bills Tracker

Track recurring bills:

1. Enter the **Bill Name**
2. Set the **Due Date**
3. Enter the **Amount**
4. Select **Frequency** from the dropdown (Monthly, Quarterly, Yearly)
5. Update **Paid Status** as you pay each bill
6. Unpaid bills are highlighted in red
7. The Dashboard shows your total bills due

## Debt Tracker

Monitor debt payoff progress:

1. Enter the **Debt Name**
2. Set the **Starting Balance**
3. Enter your **Current Balance** as you pay it down
4. Enter the **Interest Rate** (if applicable)
5. The **Payoff Progress** percentage updates automatically
6. Total debt remaining shows on the Dashboard

## Savings Tracker

Track savings goals:

1. Enter a **Goal Name** (e.g., Emergency Fund, Vacation, New Laptop)
2. Set your **Target Amount**
3. Enter the **Current Amount** saved so far
4. Set a **Target Date**
5. The **Progress** percentage updates automatically
6. Average savings progress shows on the Dashboard

## Dashboard

Your financial overview at a glance:

- **Total Income** — planned income from all sources
- **Total Expenses** — actual spending from Expense Tracker
- **Net Savings** — income minus expenses
- **Savings Rate** — percentage of income saved
- **Bills Due** — unpaid bills requiring attention
- **Debt Remaining** — total outstanding debt
- **Savings Progress** — average progress toward goals
- **Category Breakdown** — spending by category

## Common Notes

- Cells with gray background contain formulas — do not edit them
- Only edit white or green-background input cells
- Keep category names consistent across tabs for formulas to work correctly
- Clear sample data before entering your own information
