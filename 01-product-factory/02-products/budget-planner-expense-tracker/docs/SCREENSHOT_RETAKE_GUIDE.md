# Screenshot Retake Guide — Budget Planner & Expense Tracker

Retake all 4 marketplace screenshots from the live Google Sheets copy. Do not use the Excel file or generated images.

## Required Screenshots

| # | File | Content |
|---|------|---------|
| 1 | 02-dashboard.png | Full Dashboard tab |
| 2 | 03-monthly-budget.png | Monthly Budget income + expenses |
| 3 | 04-expense-tracker.png | Expense Tracker with transactions |
| 4 | 05-bills-debt-savings.png | Combined: Bills + Debt + Savings |

## General Rules

- Rename the Google Sheet first: "Budget Planner & Expense Tracker — Nivora"
- Use incognito/private window to avoid personal bookmarks
- Set browser zoom to 100%
- Crop excess browser chrome (toolbar, bookmarks, scrollbars)
- Save as PNG with maximum quality
- Target width: 1200-1920px

## 02-dashboard.png

1. Navigate to the Dashboard tab
2. Ensure all charts are visible (pie chart + bar chart)
3. Ensure the sheet tabs row is visible at the bottom
4. Crop from the title "Financial Dashboard" down to just below the "Monthly Income vs Expenses" table
5. Remove empty space at the bottom

## 03-monthly-budget.png

1. Navigate to the Monthly Budget tab
2. Focus on the **Income** section (rows 1-6) and **Expenses** section (rows 8-22)
3. Keep the column headers visible (Category, Type, Planned, Actual, Difference)
4. Remove empty rows below row 22
5. Ensure the section labels "INCOME" and "EXPENSES" are both visible

## 04-expense-tracker.png

1. Navigate to the Expense Tracker tab
2. Focus on the transaction rows
3. Open one or more dropdowns to show the data validation menu visible (Category, Payment Method, or Planned/Unplanned)
4. Remove empty lower rows
5. Keep the header row visible

## 05-bills-debt-savings.png

Combine three screenshots into one marketplace image:

1. Take separate screenshots of:
   - Bills Tracker tab (focus on bill rows)
   - Debt Tracker tab (focus on debt rows)
   - Savings Tracker tab (focus on savings rows)
2. Merge them vertically into a single image
3. Keep headers visible for each section
4. Add section labels if needed
5. Save as 05-bills-debt-savings.png
