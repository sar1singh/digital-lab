# Product Factory Marketplace Checklist

> Platform-specific requirements and differences for every marketplace Product Factory ships to.

---

## Primary Platforms (Active)

### Gumroad

| Requirement | Details |
|-------------|---------|
| Platform type | Product-first (no storefront required) |
| Thumbnail | Required — upload cover image |
| Cover image | Required — 16:9 ratio, recommended 1600×900 |
| Gallery images | Up to 10 images — upload all 6 in correct order |
| Pricing | Set in listing — supports multiple currencies |
| Payment | PayPal required for India-based sellers |
| CTA options | Buy Now, Pre-Order (limited) |
| Product file | Upload delivery ZIP |
| Test purchase | Preview mode available — test before publishing |
| SEO | Title and description indexed — optimize for search |

**Gumroad-specific checklist:**
- [ ] Cover image uploaded as thumbnail
- [ ] All 6 gallery images uploaded in correct order
- [ ] Title is SEO-friendly: `{Product Name} — Google Sheets + Excel`
- [ ] Publisher line: "by Nivora"
- [ ] Price set
- [ ] PayPal connected
- [ ] ZIP uploaded as product file
- [ ] Test purchase completed
- [ ] Description includes: What's Included, Who It's For, Benefits, FAQ, Support

---

### Payhip

| Requirement | Details |
|-------------|---------|
| Platform type | Storefront-first (brand/store name required) |
| Store setup | Create store before first listing |
| Brand name | Required — use "Nivora" |
| Cover image | Required |
| Gallery images | Supported — upload all 6 |
| Pricing | Set in listing |
| Payment | PayPal integration required |
| Product file | Upload delivery ZIP |
| Test purchase | Available in preview mode |

**Payhip-specific checklist:**
- [ ] Store created with brand name "Nivora"
- [ ] Store description matches product catalog
- [ ] PayPal integration configured
- [ ] Listing title matches Gumroad (consistent branding)
- [ ] All 6 images uploaded
- [ ] ZIP uploaded as product file
- [ ] Price matches Gumroad
- [ ] Description complete (What's Included, Who It's For, Benefits, FAQ, Support)

## Cross-Platform Consistency Rules

| Element | Must Be Consistent Across |
|---------|--------------------------|
| Brand name | Nivora on all platforms |
| Product title | Same SEO-friendly title on Gumroad + Payhip (adapt for Etsy) |
| Price | Same base price across platforms |
| Images | Same 6-image set (may need platform-specific resizing) |
| ZIP file | Same delivery ZIP on all platforms |
| Support email | workbysar1@gmail.com on all listings |
| What's Included | Same deliverables described consistently |
| Publisher line | "by Nivora" everywhere |

## Platform Migration Path

```
Product #1–3: Gumroad + Payhip only
Product #4–5: Add Etsy (if criteria met)
Product #6–10: Add Ko-fi, Creative Market (if applicable)
10+ products: Evaluate Lemon Squeezy, Sellfy, Itch.io
```

Do not open a new platform until the current ones show consistent sales or the criteria are met.
