# Listing Asset Standard

> Required marketplace image set for every Product Factory product.

---

## Required Assets (6 Images)

| # | File | Content |
|---|------|---------|
| 1 | `01-cover.png` | Product cover — brand, preview, benefits at a glance |
| 2 | `02-dashboard.png` | Dashboard or summary view with real sample data |
| 3 | `03-main-input.png` | Main input sheet — shows data entry workflow |
| 4 | `04-core-feature.png` | Core feature — the primary value driver |
| 5 | `05-secondary-features.png` | Collage of secondary features and advanced tabs |
| 6 | `06-whats-included.png` | Deliverables checklist or feature grid |

---

## Naming Convention

```
NN-description.png
```

Zero-padded sequence (01, 02, 03...). Lowercase. Hyphen-separated.

---

## Requirements

- PNG format, 1200–1920px wide
- Text legible at 280px thumbnail
- No browser chrome, bookmarks, personal data
- Brand (Nivora) on cover and where appropriate
- Real sample data — never empty sheets
- Consistent aspect ratio across all 6 images
