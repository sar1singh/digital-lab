# Product Launch Checklist

## Before Publishing

- [ ] Product file is final enough to sell.
- [ ] Example version is included.
- [ ] Instructions are included.
- [ ] Listing title is clear.
- [ ] Listing description is direct.
- [ ] Price is set.
- [ ] Preview screenshots are uploaded.
- [ ] Refund/support wording is included.
- [ ] Product catalog row is updated.

## Publishing

- [ ] Gumroad listing live.
- [ ] Payhip listing live if setup is low-friction.
- [ ] Test purchase or download flow checked if possible.
- [ ] Listing URL recorded in product catalog.

## After Publishing

- [ ] Record date published.
- [ ] Record initial price.
- [ ] Track views.
- [ ] Track sales.
- [ ] Track revenue.
- [ ] Note buyer questions.
- [ ] Decide improve, bundle, or move on.
