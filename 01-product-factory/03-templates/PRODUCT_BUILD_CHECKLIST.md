# Product Build Checklist

## Scope Lock

- [ ] Product solves one buyer problem.
- [ ] Product can be completed in 1-2 days.
- [ ] Product is a spreadsheet, calculator, template, or simple business tool.
- [ ] No SaaS.
- [ ] No code-heavy build.
- [ ] No website dependency.

## Product Contents

- [ ] Main tool/template created.
- [ ] Example data included.
- [ ] Instructions included.
- [ ] Inputs clearly labeled.
- [ ] Outputs clearly labeled.
- [ ] Assumptions visible.
- [ ] Basic error checks added where useful.
- [ ] File exported or copied into buyer-ready format.

## Quality Check

- [ ] First-time user can understand it in 5 minutes.
- [ ] Product name is clear.
- [ ] File names are clean.
- [ ] No private notes or rough drafts included.
- [ ] No unsupported legal, tax, or financial claims.

## Ready For Listing

- [ ] Preview screenshots prepared.
- [ ] What is included is clear.
- [ ] Price chosen.
- [ ] Listing draft created.
