# Marketplace Asset Standard

> Standard image set required for every Product Factory listing.

---

## Required Assets

| # | File Name | Content | Purpose |
|---|-----------|---------|---------|
| 1 | `01-cover.png` | Product cover with brand, workbook preview, key benefits | Thumbnail — primary marketplace image |
| 2 | `02-dashboard.png` | Dashboard or summary view with real sample data | Show overall product value |
| 3 | `03-primary-feature.png` | Core feature — the main reason someone buys | Primary utility demonstration |
| 4 | `04-secondary-feature.png` | Secondary workflow — supporting feature | Show depth of product |
| 5 | `05-advanced-features.png` | Less obvious features that add value | Capture undecided buyers |
| 6 | `06-whats-included.png` | Deliverables checklist or feature grid | Answer "what do I get?" |

---

## Naming Convention

```
NN-feature-description.png
```

Where NN is a zero-padded sequence number (01, 02, 03...).

Examples:
- `01-cover.png`
- `02-dashboard.png`
- `03-primary-feature.png`

---

## Image Requirements

| Property | Specification |
|----------|---------------|
| Format | PNG only |
| Width | 1200–1920px |
| Aspect ratio | Maintain consistent ratio across set |
| Text | Legible at 280px thumbnail width |
| Brand | Nivora must appear on cover and where appropriate |
| Data | Real sample data — no empty sheets |
| Chrome | No browser UI, bookmarks, tabs |

---

## Order on Marketplace

Upload images in this exact sequence:

| Position | Asset | Why |
|----------|-------|-----|
| 1st | Cover | Primary thumbnail — drives click-through |
| 2nd | Dashboard | First impression of product value |
| 3rd | Primary feature | Core utility confirmation |
| 4th | Secondary feature | Supports purchase decision |
| 5th | Advanced features | Converts hesitant buyers |
| 6th | What's Included | Final confirmation before checkout |

---

## Generation Timing

Generate all 6 assets during the **Packaging** phase, before listing creation. Do not defer to after listing is created.
