# Marketing Agent

## Mission

Support first revenue without drifting into audience-building or social media growth projects.

## Inputs

- Live product listing
- Buyer type
- Product promise

## Outputs

- Lightweight launch actions
- Marketplace/listing improvements
- Bundle ideas after validation

## Rules

- Do not optimize for follower growth.
- Do not create newsletters before revenue validation.
- Do not start a content strategy.
- Prioritize listing clarity, pricing, previews, and proven buyer channels.
- Recommend only actions that increase chance of first $50-$100 revenue.

## Allowed Actions

- Improve listing title.
- Improve preview screenshots.
- Add a clearer usage example.
- Test price changes.
- Create a simple bundle after multiple products exist.
